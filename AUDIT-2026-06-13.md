# Plugin Marketplace Audit ‚Äî 2026-06-13

Three independent reviewers audited all 14 plugins for currency (newer
models/APIs/CLIs), obsolescence, improvements, and bugs:

- **Claude** ‚Äî 5 parallel domain agents, verified against **live docs** (web + context7).
- **Codex** (CLI 0.139) ‚Äî whole-repo read-only pass, told the private CLIs are valid.
- **GLM-5.2** (via the `consult` plugin) ‚Äî whole-source pass, training-knowledge based.

**Confidence:** findings tagged `[3/3]` / `[2/3]` = number of reviewers that
independently raised it. `[verified]` = I confirmed it directly in this session.

**Private-CLI caveat:** `cloudflare`, `unifi`, `bambu` wrap the author's
**private** CLIs. All "CLI does not exist / 404" findings were **dropped** as
false alarms. Only CLI-agnostic findings (credentials, env-var naming,
validation) are kept for those three.

## Priority action list (cross-plugin)

### P0 ‚Äî broken right now
1. **coderabbit** ‚Äî `coderabbit --prompt-only` flag was **removed**; correct form is
   `coderabbit review --agent -t all|committed|uncommitted`. Fails on every run. `[3/3] [verified CLI 0.6.0]`
   Files: `agents/cr-reviewer.md:71`, `skills/cr-local/SKILL.md:42`, `commands/local.md`.
2. **multi-ai-review + fork-terminal** ‚Äî `codex --full-auto` **removed** in Codex 0.139;
   use `codex exec --dangerously-bypass-approvals-and-sandbox`. `[3/3] [verified]`
   Files: `multi-ai-review/scripts/cli_configs.py:29`, `fork-terminal/.../cookbook/codex-cli.md`.

### P1 ‚Äî security
3. **damage-control** ‚Äî hook `timeout: 5000` is **seconds** ‚Üí a stuck hook hangs ~83 min.
   Set to `5`‚Äì`10`. `[verified]` (`hooks/hooks.json:10,...`)
4. **damage-control** ‚Äî `zeroAccessPaths` is advertised to block secret reads, but only
   `Bash/Edit/Write` are hooked ‚Äî **`Read` is not**, so secrets are still readable. Add a
   `Read` (and `MultiEdit`) PreToolUse hook. `[2/3 Claude+Codex]`
5. **fork-terminal** ‚Äî `safe_model = shlex.quote(model).strip("'")` strips the quotes
   `shlex.quote` just added, **re-introducing shell injection** if `model` carries
   metacharacters. `[GLM] [needs verify]` (`tools/spawn_session.py:47`)
6. **fork-terminal / multi-ai-review** ‚Äî `--dangerously-skip-permissions` /
   Codex sandbox-bypass / Gemini `-y` are the **default** path. Make sandboxed/interactive
   the default; require explicit opt-in for bypass. `[2/3 Codex+GLM]`
7. **ultimate-statusline** ‚Äî `install.sh` overwrites `~/.claude/settings.json` `statusLine`
   **non-interactively with no backup**; the custom-command widget executes arbitrary
   config-supplied commands. `[Codex]` (`install.sh:128`, `widgets.sh:722`)

### P2 ‚Äî model / API currency
8. **imagegen** ‚Äî default `gpt-image-2` mismatches docs/fallbacks: README config table says
   `gpt-image-1`, `compare.py` defaults `gpt-image-1`, `estimate_cost()` has no `gpt-image-2`
   price (returns "Unknown"). `gpt-image-2` is generations-only and **rejected by the edits
   endpoint** + can't do `background=transparent`. Align defaults, guard transparent, add
   `chatgpt-image-latest` alias. `[3/3]`
9. **video-gen** ‚Äî Sora `/v1/videos` **shuts down 2026-09-24** with no runtime guard; `1:1`
   silently maps to `1280x720` landscape while docs claim square; `sora-status.sh` expects a
   JSON URL where the API now returns MP4 content; missing `veo-3.1-lite`. `[3/3]`
10. **multi-ai-review / fork-terminal** ‚Äî `gemini-3-pro-preview` ‚Üí `gemini-3.1-pro-preview`
    (or `gemini-pro-latest` alias); unify the contradictory Codex/Claude model defaults across
    plugins. `[2/3]`

### P3 ‚Äî modernization / polish
11. **fork-terminal / multi-ai-review** ‚Äî Claude Code now has **background agents, agent
    teams, `--worktree`**; prefer those for Claude workers, keep these plugins for the
    cross-CLI consensus angle. `[Codex]`
12. **ui-ux-pro-max** ‚Äî description claims **"integrates shadcn/ui MCP" but there is no MCP
    wiring**; Svelte data mixes contradictory Svelte 4/5 syntax; React/Tailwind/Next anchored
    to old majors; committed `__pycache__` (py3.14); BM25 returns top rows even at **zero
    score**, so irrelevant guidance looks authoritative. `[3/3 across items]`
13. **ultimate-statusline** ‚Äî recomputes `git-worktree` and context-% via subprocess when CC
    now supplies them in the statusline JSON; `cache.sh` is dead code (never invoked);
    `mapfile` breaks on macOS bash 3.2; missing `effort`/`thinking`/`rate_limits` widgets. `[2/3]`
14. **audio-feedback** ‚Äî `task-complete.sh` reads `CLAUDE_SUBAGENT_*` **env vars** that may be
    empty in CC 2.x; it should parse the hook **stdin JSON**. Kokoro version unpinned (v0.5.0
    changed voice/model IDs); `test.md` pipes to `afplay -` which can't read stdin. `[3/3]`
15. **consult** ‚Äî add retry/backoff (429/5xx) + a total-timeout ceiling; the optional Stop hook
    relies on `${CLAUDE_PLUGIN_ROOT}` inside `.claude/settings.json` (may not resolve) and sends
    transcript+diff to a third party (add a consent/redaction note). `[Codex + self]`
